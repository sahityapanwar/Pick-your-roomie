# 🏠 GSV Hostel Roommate Selection Portal

> **A centralized web platform that enables second-year GSV College
> students to find roommates, form groups, and organize hostel
> accommodation in a transparent and hassle-free manner.**

------------------------------------------------------------------------

## 📌 Overview

The **GSV Hostel Roommate Selection Portal** is a full-stack web
application developed to simplify the roommate selection process for
students moving into the GSV College hostel.

Instead of relying on scattered communication through WhatsApp groups or
personal messages, the portal provides a centralized platform where
students can:

-   Browse all eligible hostel students
-   Find roommates based on their current status
-   Form roommate groups
-   Send and manage invitations
-   Track roommate requests
-   Complete their trio before moving into the hostel

The application ensures transparency, prevents duplicate roommate
selections, and makes the entire process organized and efficient.

------------------------------------------------------------------------

## 🎯 Problem Statement

Every academic year, students struggle to coordinate roommate selection
using multiple communication channels.

The portal solves problems such as:

-   Difficulty finding available roommates
-   Duplicate or conflicting roommate selections
-   Lack of transparency
-   Manual coordination through WhatsApp

## ✨ Features

### 👤 Student Authentication

-   Roll Number based login
-   Temporary password authentication
-   Personal student account

### 📚 Student Directory

-   Browse all eligible students
-   Search by name or roll number
-   Filter by roommate status
-   View student profiles

### 🤝 Roommate Invitation System

-   Invite another student to form a duo
-   Accept or reject invitations
-   Receive roommate notifications

-   Create roommate duos
-   Accept one additional roommate

-   View all roommate groups

-   Trio formation
-   Profile updates

### 📊 Dashboard

Live statistics including:

-   Total Students
-   Students Looking for Roommates
-   Total Duos
-   Completed Trios
-   Pending Requests

### 👨‍💼 Admin Panel

-   Add/Edit/Delete students
-   Manage groups
-   Manage requests
-   Export student data
-   View overall statistics

------------------------------------------------------------------------

## 🚀 How It Works

1.  Log in using your college roll number.
2.  Browse the student directory.
3.  Invite another student to form a duo.
4.  Accept or reject roommate invitations.
5.  Complete your trio by adding a third roommate.
6.  Once your trio is finalized, move into the hostel and choose your
    preferred room on a **first-come, first-served basis**.

## ⚙️ Installation

``` bash
git clone https://github.com/yourusername/GSV-Hostel-Roommate-Selection.git
cd GSV-Hostel-Roommate-Selection/backend
pip install -r requirements.txt
python app.py
```

Then open `frontend/index.html` in your browser or use the VS Code Live
Server extension.
